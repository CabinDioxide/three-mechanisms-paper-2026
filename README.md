# 中国未现代化的三机制：朋党之锁的跨王朝定量检验

张笑宇 · 工作论文 v0.4 · 2026-04-28

## 论文核心

中国从未发展出能够约束最高权威的、由非血缘成员构成的合法政治主体。这一缺失既不是物质条件的缺失，也不是制度想象的缺失——它是一个话语层面的结构性缺失。

本文回答一个问题：中文政治词汇中存在哪些能指代合法横向联合的范畴？哪些被合法化、哪些被禁绝、哪些时序振荡？这一分布如何在跨王朝中演化？

## 方法

跨越先秦至晚清两千五百年、规模 1.04 亿字的古典中文语料库，独立训练十个分时段的 word2vec 词向量模型，对约 130 个候选词（按七层语义结构分组）做全谱系扫描。

## 主要发现

中文话语场域发展出了 *conjuratio*（誓约联合）和 *communitas*（共同体）的合法范畴——盟、誓、契、约、社、乡党、宗族都在合法语义场中——但作为合法政治主体的 *universitas*（法人 / 个体被吞没的合法集合）在中国从未结晶。这才是"中国为何未发展出宪政"的精确答案。
